1º Desafio Evolux.

Escrever uma aplicação web em Python usando o framework Flask (http://flask.pocoo.org/)

  1. A aplicação é um cadastro de bares. Crie um formulário de cadastro com os seguintes campos:

    2. Nome do bar:
    Endereço:
    Horário de Abertura:
    Horário de Abertura:
    Especialidade da Casa:

  2. Ao submeter o formulário, os dados enviados pelo usuário devem ser submetidos via POST, capturados e renderizados na tela.
  3. Salve os dados submetidos no formulário em um arquivo de texto. Você pode escolher o formato, porém sugerimos uso de yml
